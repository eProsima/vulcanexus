Vulcanexus HRI Speech-to-Text (STT) Tutorial
=============================================

Quickview
---------

In this tutorial, we will explore the process of implementing speech-to-text capabilities within the Vulcanexus framework.
We will cover the necessary steps to set up the environment, integrate the speech-to-text model, and visualize the results in RViz2.

Prerequisites
-------------

Before starting this tutorial, ensure you have the following prerequisites:

- A working installation of Vulcanexus.
- Familiarity with ROS 2 concepts and tools.
- A microphone or a compatible audio input device.

Vulcanexus HRI Face Detection Tutorial
======================================

Quickview
---------

In this tutorial, we will explore the process of implementing face detection capabilities within the Vulcanexus framework.
We will cover the necessary steps to set up the environment, integrate the face detection model, and visualize the results in RViz2.

Prerequisites
-------------

Before starting this tutorial, ensure you have the following prerequisites:

- A working installation of Vulcanexus.
- Familiarity with ROS 2 concepts and tools.
- USB camera or a compatible video input device.

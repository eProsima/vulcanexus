VulcanAI Terminal
=================

.. note::
   VulcanAI is currently in active development, and new features and improvements are being added regularly.
   Current version is in Beta stage.

The VulcanAI textual terminal provides interactive command execution, command history, and keyboard shortcuts to speed up common operations.

Text Selection and Clipboard
----------------------------

You can select any area in the terminal by left-clicking and dragging over the desired text.
Once text is selected, press ``F3`` to copy the selected area to the system clipboard.

Keyboard Shortcuts
------------------

The following shortcuts are the most useful. Check for all the shortcuts in the help message!

.. list-table::
   :header-rows: 1
   :widths: 30 70

   * - Shortcut
     - Description
   * - ``F2``
     - Display the terminal help message.
   * - ``F3``
     - Copy the currently selected terminal area to the clipboard.
   * - ``Ctrl+R``
     - Start reverse search through command history.
   * - ``Ctrl+L``
     - Clear the terminal quickly.
   * - ``Tab``
     - Autocomplete the command while typing.
